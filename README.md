# Extension Assunção et al. (2023) - Optimal Environmental Targeting

This repository contains the replication code for the study on the impact of the Priority List on deforestation in the Brazilian Amazon.

## Project Structure
- `01_descriptives_deforestation.R`, `01_descriptive_figure2.R`: Analysis of deforestation trends.
- `02_DiD.R`: Difference-in-Differences estimations.
- `03_heterogeneity.R`: Extensions.

## How to run
Run the scripts in chronological order (01 to 03).
